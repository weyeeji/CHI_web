01

为什么需要这个实验系统？ Why does this experiment system exist?

LLM-based agents 正在从一对一助手转向多个 agents 组成的团队。用户面对的不再只是单个 assistant 的输出质量，而是一个 agent team 是否分工清楚、能否产生有用分歧、是否能把探索与批判综合成可用方案。 LLM-based agents are shifting from one-to-one assistants toward teams of visible agents. Users no longer evaluate only a single assistant's answer quality; they also judge whether an agent team has clear division of labor, useful disagreement, and effective synthesis.

OpenClaw 等系统中的 SOUL.md 说明 personality 已经可以被写成稳定配置：voice、stance、style、boundaries 和默认行为倾向都可以被设计。但现有研究更多关注 single agent personality，缺少对 agent team 中 personality 如何影响协作体验的研究。 Systems such as OpenClaw make personality configurable through files like SOUL.md, where voice, stance, style, boundaries, and default behavioral tendencies can be authored. Existing research still focuses more on single-agent personality than on how personality operates inside an agent team.

1:1 assistant 单个模型的稳定表现、语气、可信度。 Stable behavior, tone, and trust for one assistant.
Agent team 多个角色之间的互补、分歧、综合和协调负担。 Complementarity, disagreement, synthesis, and coordination cost across roles.
Research gap SOUL/personality 是否成为团队层面可感知、可评价、可设计的变量。 Whether SOUL/personality becomes a perceivable, evaluable, designable team-level variable.
